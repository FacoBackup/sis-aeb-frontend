1. Frontend
   ====================
   Source code para as duas interfaces do sistema:

    1.A - ADMINISTRATION:
        Administração do sistema. Contém interface para criação e atualização de entidades, suporte a autenticação e acesso a outras informações.
        
    1.B - MAIN:
        Contém a interface para os Ramais e estruturas organizacionais.
